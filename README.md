# Walwal Cards 🃏🍻

Walwal Cards is a fun, party/drinking card game built with React Native and Expo. It allows users to play an interactive card game directly on their mobile devices (iOS/Android) or on the Web.

## Features ✨

- **Cross-Platform:** Built with Expo, supporting iOS, Android, and Web out of the box.
- **Modern Styling:** Utilizes [NativeWind](https://www.nativewind.dev/) (Tailwind CSS) for responsive and sleek UI components.
- **Animations:** Smooth card transitions powered by React Native Reanimated.
- **Backend integration:** Integrated with [Supabase](https://supabase.com/) for managing cards, game states, or real-time data.

## Tech Stack 🛠️

- **Framework:** [React Native](https://reactnative.dev/) / [Expo](https://expo.dev/) (Expo Router)
- **Language:** TypeScript
- **Styling:** NativeWind / Tailwind CSS
- **Backend:** Supabase
