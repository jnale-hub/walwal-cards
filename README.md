# Walwal Cards

A fun Pinoy drinking card game app built with Expo.

## How to Run

1. **Install**
   ```bash
   npm install
   ```

2. **Start**
   ```bash
   npx expo start
   ```

3. **Play**
   - Scan the QR code with your phone (using Expo Go).
   - Or press `a` for Android emulator.
   - Or press `i` for iOS simulator.

## Project Structure

- **app/**: The main screens of the app.
- **src/components/**: Reusable parts like buttons and cards.
- **src/constants/**: Game data and styles.
